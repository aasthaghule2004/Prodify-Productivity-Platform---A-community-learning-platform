**Prodify** is a productivity-driven study platform built to help students organize, manage, and share study materials efficiently.
The goal of the platform is to create a **community-centered learning environment** where users can store notes, share resources, and interact with other learners.

---

# Features

**Study Material Management**

* Upload and organize study notes
* Structured resource management

**User Authentication**

* Secure user registration and login
* Personal dashboards for users

**Community-Based Learning**

* Option to **share notes with others**
* Ability to keep notes **private or public**

**Productivity Focused**

* Clean interface designed for focused studying
* Easy navigation for quick access to materials

# ⚙️ Installation Guide

### 1️⃣ Clone the repository

```
git clone https://github.com/your-username/prodify.git
```

### 2️⃣ Navigate into the project

```
cd prodify
```

### 3️⃣ Create a virtual environment

```
python -m venv venv
```

### 4️⃣ Activate virtual environment

**Windows**

```
venv\Scripts\activate
```

**Mac/Linux**

```
source venv/bin/activate
```

### 5️⃣ Install dependencies

```
pip install -r requirements.txt
```

### 6️⃣ Apply migrations

```
python manage.py migrate
```

### 7️⃣ Run the development server

```
python manage.py runserver
```

Open in browser:

```
http://127.0.0.1:8000/
```
